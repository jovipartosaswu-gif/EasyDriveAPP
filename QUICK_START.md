# Quick Start Guide

## What to Do Now

### 1. Build and Run (That's It!)
```
1. Open Android Studio
2. Click the green play button (Run)
3. Wait for build to complete
4. App launches on your device
```

### 2. Test Adding a Car with Image
```
1. Login as Agency (or create agency account)
2. Click "Add New Car"
3. Fill in car details
4. Click "Select Image"
5. Choose any image from gallery
6. Click "Add Car"
7. Success! ✅
```

### 3. Verify Images Display
```
1. Go to "Manage Cars" - image should show ✅
2. Logout and login as Renter
3. Home screen shows car with image ✅
4. Click "View Details" - image shows ✅
```

## Optional: Enable Cloud Storage Later

If you want images in the cloud (accessible from any device):

1. Go to https://console.firebase.google.com/
2. Select your project
3. Click "Storage" → "Get Started"
4. Choose "Start in test mode"
5. Done!

Images will then upload to cloud automatically, with local fallback if needed.
